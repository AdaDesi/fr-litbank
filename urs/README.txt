Les fichiers dans ce répertoire .urs sont des fichiers zippés.

mv *.usr *.zip

puis

unzip *.zip

créé un répertoire avec l nom du roman, puis 2 sous-répertoires :

- texts qui contient le *.xml -> recopié dans le répertoire xml du dépot

- annotations qui contient le -urs.xml -> recopié dans le répertoire urs-xml du dépôt

Note : certains ont été renommés pour plus de clarté (dd.xml -> diableaucorps.xml)
